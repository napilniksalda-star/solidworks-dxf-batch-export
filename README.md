# Экспорт развёрток листового металла в DXF по шаблону — макрос SolidWorks

Макрос для SolidWorks: открывает деталь из листового металла, проходит по **всем её конфигурациям** и выгружает развёртку каждой в отдельный DXF (для лазерной/плазменной резки, раскроя).

Имя файла собирается по **шаблону**, который ты задаёшь в самой детали. Поэтому один макрос работает для **любых** деталей — пластин, уголков, профилей и т.п. Гнутые детали разворачиваются в плоскую заготовку автоматически.

---

## Требования

- SolidWorks с поддержкой листового металла.
- Деталь `.sldprt` из листового металла с несколькими **конфигурациями** (обычно через таблицу параметров).
- Русский интерфейс SolidWorks (имена размеров вида `D1@Эскиз1`). При другом языке имена в шаблоне будут свои.

## Скачать

- Кнопка **`Code` → `Download ZIP`** на странице репозитория, **или**
- `git clone https://github.com/napilniksalda-star/solidworks-dxf-batch-export.git`

Нужен один файл: **`src/TemplateDxfExport.bas`**.

## Установка макроса в SolidWorks

1. **`Сервис → Макрос → Создать…`** (Tools → Macro → New), сохрани новый макрос (например `DxfExport.swp`). Откроется редактор VBA.
2. Открой `src/TemplateDxfExport.bas` в текстовом редакторе (VS Code / Блокнот), выдели **весь** текст, скопируй.
3. В редакторе VBA открой модуль (слева, напр. `Module1`), выдели всё (`Ctrl+A`) и вставь.
   - ⚠️ Вставляй именно **копированием-вставкой**, а не через `File → Import` — так кириллица не испортится. Строки `Attribute` в файле нет, он готов к вставке.
4. **`Debug → Compile`** — ошибок быть не должно.

## Подготовка детали (3 шага)

**1. Деталь** — листовой металл с несколькими **конфигурациями** (через таблицу параметров). Одна строка таблицы = одна конфигурация = одна заготовка = один DXF.

**2. Таблица параметров — два вида столбцов:**
- **Размеры** детали по полному имени размера, как обычно: `D1@Эскиз1`, `D2@Эскиз1`, `D2@Базовая кромка1`, `Толщина@Листовой металл`.
- **Количество и любые НЕразмерные данные** — как **свойство**, обязательно с префиксом `$PRP@` (в русском интерфейсе — `$СВОЙСТВО@`):
  `$PRP@Количество`. Без префикса SolidWorks столбец не примет.

**3. Шаблон имени** — `Файл → Свойства → вкладка «Настройки»` → добавь свойство **`Шаблон_имени`** = будущее имя файла с подстановками (см. ниже).

## Шаблон имени

Значение свойства `Шаблон_имени` — это имя файла, где меняющиеся части записаны в **фигурных скобках** `{ }`:

- `{D1@Эскиз1}` — есть `@` → подставляется значение **размера** из геометрии (в мм).
- `{Количество}` — нет `@` → подставляется значение **свойства** (сначала конфигурации, потом файла).

⚠️ Только фигурные скобки `{ }`, **не** кавычки `" "`. Кавычки — это язык самого SolidWorks, он вычислит выражение один раз, и у всех файлов будет одинаковое имя. **Проверка:** в колонке «Вычисленное значение» свойства должны остаться **скобки**, а не числа.

## Примеры

**Уголок** (`Шаблон_имени`):
```
Уголок {D1@Эскиз1}x{D2@Эскиз1}x{D2@Базовая кромка1}_Qty_{Количество}
```

| конфиг | D1@Эскиз1 | D2@Эскиз1 | D2@Базовая кромка1 | $PRP@Количество |
|---|---|---|---|---|
| 1000 | 50 | 100 | 1000 | 10 |
| 1010 | 60 | 120 | 1010 | 11 |

→ `Уголок 50x100x1000_Qty_10.dxf`, `Уголок 60x120x1010_Qty_11.dxf`

**Пластина** (`Шаблон_имени`):
```
Plate_{D1@Эскиз1}x{D2@Эскиз1}x{Толщина@Листовой металл}_Qty_{Количество}
```
→ `Plate_52x300x1.5_Qty_2.dxf`

## Запуск

1. Открой и **сохрани** деталь.
2. Запусти макрос: **`Сервис → Макрос → Запустить…`** (или `F5` в редакторе).
3. Выбери папку для DXF.
4. В конце — отчёт: **выгружено / пропущено / ошибок**. После работы деталь возвращается на исходную конфигурацию.

## Имя файла

- Подстановка делается **по каждой конфигурации** отдельно.
- Числа: разделитель дробной части — точка, лишние нули убираются (`2.0 → 2`).
- При совпадении имён к файлу добавляется имя конфигурации (затем `_2`, `_3`) — **перезаписи нет**.

## Если что-то не так

| Сообщение / симптом | Причина |
|---|---|
| «Открой деталь!» | Нет активной детали |
| «Это не деталь!» | Открыт не `.sldprt` |
| «Сначала сохрани файл!» | Деталь не сохранена на диск |
| «Добавь … свойство `Шаблон_имени`» | У детали нет шаблона имени |
| «В шаблоне нет подстановок `{ }`» | Шаблон записан кавычками `"…"`; используй `{ }` |
| Во всех DXF одинаковое имя | То же — шаблон в кавычках. «Вычисленное значение» должно показывать скобки, не числа |
| SolidWorks ругается на столбец свойства | Добавь префикс `$PRP@` / `$СВОЙСТВО@` (напр. `$PRP@Количество`) |
| «ПРОПУСК (нет листового металла)» | В конфигурации нет элемента листового металла / развёртки |
| «ОШИБКА (шаблон/размер)» | Размер из `{…@…}` не найден — проверь имя размера в шаблоне |

## Настройка

В начале модуля:
```vba
Private Const TEMPLATE_PROP As String = "Шаблон_имени"   ' имя свойства файла с шаблоном
```
Если хочешь другое имя свойства для шаблона — поменяй здесь.
